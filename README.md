# 🧮 Python CLI Calculator

A simple command-line calculator built with Python. It performs basic arithmetic operations: addition, subtraction, multiplication, and division.

## 📦 Requirements

- Python 3.x installed

## ▶️ How to Run

1. Save the code in a file called `calculator.py`.

2. Open your terminal or command prompt and navigate to the file directory.

3. Run the calculator:

```bash
python calculator.py
